# Matcha Mall

말차를 주제로 한 쇼핑몰 형태의 웹사이트 포트폴리오 프로젝트입니다.

## 기능

* 상품 목록 표시
* 상품 카드 UI
* 상품 상세 정보
* 장바구니 인터페이스
* 반응형 웹 디자인
* 모던 쇼핑몰 레이아웃

## 사용 기술

* HTML
* CSS
* JavaScript

## 배포 링크

https://lakekimhandsome.github.io/matcha-mall/

## 목적

쇼핑몰 웹사이트의 사용자 인터페이스와 상품 탐색 경험을 구현하기 위해 제작한 프로젝트입니다. 실제 결제 기능은 포함되어 있지 않으며, 프론트엔드 개발 역량을 보여주기 위한 포트폴리오 프로젝트입니다.

## 핵심 개념

* E-Commerce UI Design
* Responsive Web Design
* Product Card Components
* Shopping Cart Interface
* User Experience (UX)

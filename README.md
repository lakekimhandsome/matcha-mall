# 말차몰 (Matcha Mall)

프리미엄 말차 쇼핑몰 — vanilla HTML/CSS/JS

## 구성

- **상품 목록** — 카테고리 필터 (파우더 / 음료 / 간식)
- **베스트 상품** — 인기 상품 섹션
- **상품 상세** — `product.html?id=1` 형식
- **장바구니** — 슬라이드 패널, localStorage 저장

## 실행

```bash
cd matcha-mall
open index.html
```

또는 Live Server 등으로 `index.html`을 열면 됩니다.

## 기술

- HTML5 / CSS3 / Vanilla JavaScript
- localStorage 장바구니
- 반응형 레이아웃
